# Reports

The completed public policy research brief will be tracked here in Markdown. A
disclosure-reviewed PDF will be published as a release and archive asset, not
committed to Git. No brief exists yet because Lumen researcher access is pending
and data collection has not begun.

Each release will identify its protocol version, query-manifest version,
analysis-code revision, publication date, and archival DOI when available. It
will describe each analyzed field using the semantics and missingness rules in
the approved manifest, distinguish requested or reported action from action
actually taken and Lumen record state, and disclose when privacy-driven
minimization requires a claim to be omitted or weakened.
