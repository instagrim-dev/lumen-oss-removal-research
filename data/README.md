# Data Directory

Only disclosure-reviewed derived data belongs in this directory.

Permitted public artifacts include disclosure-reviewed aggregate tables, a
documented data dictionary, and a minimized source index. The source index may
contain only a Lumen notice identifier and the bounded, derived inclusion basis
defined by the approved query manifest. It must not contain raw or bulk URLs,
raw notice text, raw query matches, or unnecessary personal information.

Privacy and data minimization take precedence over auditability. Claims that
cannot be supported within this public-data boundary must be omitted or
weakened. Raw API responses, unreviewed notice fields, credentials, and bulk
complained-of URLs must remain outside version control.

The ignored `data/raw/`, `data/private/`, and `data/cache/` paths are local
working areas and are not publication surfaces.
